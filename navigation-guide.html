Website Navigation Guide

A simple guide for navigating the Empowering Youth website. For client use and maintenance reference.

Main Navigation

  • 🏠 Home - Overview, latest announcements, featured programs
  • About - Mission, vision, our journey, leadership
  • Programs - All initiatives and services
  • Events - Latest news and upcoming events
  • Gallery - Photos and videos from activities
  • Get Involved - Volunteer, donate, partner forms
  • Contact - Location, phone, email, form

Key Features

  • ✅ Dynamic content from JSON files (events, news, programs)
  • ✅ Image-optimized (responsive srcset, lazy loading)
  • ✅ Mobile-responsive design
  • ✅ Netlify forms (contact, volunteer, newsletter)
  • ✅ Social media links in footer (Facebook, Instagram, TikTok)
  • ✅ WhatsApp & Donate floating buttons

Adding Latest Announcements (Home Page)

Edit data/news.json, add new entry:

{
  "title": "New Event",
  "slug": "new-event",
  "date": "2025-01-01T09:00:00+02:00",
  "excerpt": "Short description...",
  "image": "assets/gallery/new-image.jpg"
}

Images auto-display on home page. Link to events.html#slug
